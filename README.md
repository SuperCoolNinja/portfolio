# portfolio
Personnal Portfolio.

## Stacks : 
* ReactJS
* Typescript
* TailwindCSS

## Installation : 
### For this project I use **PNPM** as package manager: https://pnpm.io/motivation
> * pnpm install 

###  Run the project : 
> * pnpm dev


### Build the project :
> * pnpm build
